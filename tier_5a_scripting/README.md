# Tier 5a — Scripting & Web Frameworks (Weeks 9-10)

**Languages:** Ruby, PHP, Elixir
**Patterns:** Convention-over-Configuration, LiveView, OTP Supervision Trees

## Samples

### ruby_fundamentals.rb

Ruby fundamentals: duck typing, everything-is-an-object (integers, nil, booleans), blocks/procs/lambdas, singleton classes (eigenclasses), mixins via modules, method_missing, open classes, pattern matching (Ruby 3+), Enumerator/Fiber, and metaprogramming.

```bash
ruby ruby/samples/ruby_fundamentals.rb
```

### php_fundamentals.php

PHP 8.1+ fundamentals: typed properties, enums (backed and pure), match expressions, named arguments, fibers, readonly classes, union/intersection types, attributes, first-class callables, null coalescing, and arrow functions.

```bash
php php/samples/php_fundamentals.php
```

### elixir_fundamentals.exs

Elixir fundamentals on the BEAM VM: atoms, pattern matching, pipe operator, immutable data, processes (spawn/send/receive), GenServer, supervision trees, protocols, behaviours, comprehensions, and Stream for lazy evaluation.

```bash
elixir elixir/samples/elixir_fundamentals.exs
```
