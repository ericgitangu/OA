# Tier 2 — JVM Languages (Weeks 3-4)

**Languages:** Java, Kotlin, Scala
**Patterns:** CQRS, Hexagonal Architecture, Reactive Streams

## Samples

### JavaFundamentals.java

Java 21 fundamentals: records, sealed interfaces, pattern matching in switch, virtual threads, structured concurrency, streams, lambdas, CompletableFuture, and the Collections API. Demonstrates modern Java's approach to algebraic data types via sealed permits.

```bash
java java/samples/JavaFundamentals.java
```

### KotlinFundamentals.kt

Kotlin fundamentals: sealed classes, data classes, enum classes, object singletons, extension functions, null safety, coroutines, inline/reified generics, delegation, and DSL builders. Shows how Kotlin simplifies patterns that are verbose in Java.

```bash
kotlin kotlin/samples/KotlinFundamentals.kt
```

### ScalaFundamentals.scala

Scala 3 fundamentals: sealed traits, case classes, enum types, given/using (implicits), extension methods, opaque types, pattern matching, Futures, and functional composition. Demonstrates Scala's unified approach to OOP and FP.

```bash
scala scala/samples/ScalaFundamentals.scala
```
