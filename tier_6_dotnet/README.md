# Tier 6 — .NET Ecosystem (Weeks 13-14)

**Languages:** C#, F#
**Patterns:** Clean Architecture, ADTs via Discriminated Unions, LINQ Pipelines

## Samples

### CSharpFundamentals.cs

C# fundamentals on the CLR: value vs reference types, records, pattern matching, async/await with Task, LINQ, generics with constraints, nullable reference types, channels, span/memory, init-only setters, and top-level statements.

```bash
dotnet script csharp/samples/CSharpFundamentals.cs
```

### FSharpFundamentals.fs

F# fundamentals: let bindings (immutable by default), pipe operator, discriminated unions, record types, pattern matching, computation expressions (async), active patterns, type providers, and Hindley-Milner type inference.

```bash
dotnet fsi fsharp/samples/FSharpFundamentals.fs
```
