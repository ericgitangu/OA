# LeetCode Solutions

Organized by pattern, not language. Each directory contains solutions across multiple languages for cross-language comparison.

## Patterns

| Directory | Key Problems |
|-----------|-------------|
| `arrays_and_hashing/` | Two Sum, Group Anagrams, Top K Frequent |
| `two_pointers/` | Valid Palindrome, 3Sum, Container With Most Water |
| `sliding_window/` | Best Time to Buy/Sell, Longest Substring |
| `stacks_and_queues/` | Valid Parentheses, Min Stack |
| `linked_lists/` | Reverse Linked List, Merge Two Sorted |
| `trees_and_tries/` | Invert Binary Tree, Word Search II |
| `graphs/` | Number of Islands, Course Schedule |
| `dynamic_programming/` | Climbing Stairs, Coin Change, LCS |
| `greedy/` | Jump Game, Task Scheduler |
| `backtracking/` | Subsets, Permutations, N-Queens |
| `bit_manipulation/` | Single Number, Counting Bits |
| `intervals/` | Merge Intervals, Meeting Rooms |
| `math_and_geometry/` | Rotate Image, Spiral Matrix |

## Naming Convention

```
<pattern>/<problem_name>.<ext>
# e.g., two_pointers/valid_palindrome.py
# e.g., two_pointers/valid_palindrome.rs
```
