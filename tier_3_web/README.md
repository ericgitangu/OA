# Tier 3 — Web & Full-Stack (Weeks 5-6)

**Languages:** JavaScript, TypeScript
**Patterns:** Micro-Frontends, Server Components, Edge Functions

## Samples

### js_fundamentals.js

JavaScript fundamentals on Node.js: let/const scoping, template literals, destructuring, spread/rest, closures, prototypal inheritance, classes, iterators, generators, async/await, Promises, Proxy/Reflect, WeakMap/WeakRef, and the event loop model.

```bash
node javascript/samples/js_fundamentals.js
```

### ts_fundamentals.ts

TypeScript fundamentals: structural typing, type erasure, interfaces vs type aliases, generics with constraints, mapped/conditional types, discriminated unions, utility types (Partial, Pick, Omit), template literal types, and strict null checking.

```bash
npx tsx typescript/samples/ts_fundamentals.ts
```
